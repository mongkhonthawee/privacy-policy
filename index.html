ขอขอบคุณที่ใช้แอปพลิเคชันของเรา ("แอปพลิเคชัน") เราให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้งานอย่างยิ่ง โดยนโยบายนี้จะอธิบายว่าข้อมูลใดถูกเก็บรวบรวม ใช้อย่างไร และเก็บรักษาไว้อย่างไร

1. ข้อมูลที่เราเก็บรวบรวม

เมื่อคุณใช้แอปของเรา โดยเฉพาะหากมีการเชื่อมต่อผ่านบัญชี Facebook เราอาจเก็บข้อมูลดังต่อไปนี้ (โดยได้รับความยินยอมจากคุณ):

  • ชื่อและนามสกุล
  • อีเมล
  • รูปโปรไฟล์
  • Facebook User ID
  • ข้อมูลจากอุปกรณ์ เช่น ประเภทอุปกรณ์และระบบปฏิบัติการ
  • พฤติกรรมการใช้งานแอป

2. วิธีการใช้ข้อมูล

ข้อมูลที่เรารวบรวมจะถูกนำไปใช้เพื่อ:

  • ให้บริการและแสดงผลฟีเจอร์ต่างๆ
  • ปรับปรุงประสบการณ์การใช้งานของผู้ใช้
  • ตอบคำถามหรือข้อเสนอแนะจากผู้ใช้
  • ส่งข่าวสารหรือการแจ้งเตือนที่เกี่ยวข้อง (หากคุณยินยอม)

3. การแบ่งปันข้อมูล

เราไม่ขายหรือแบ่งปันข้อมูลส่วนตัวของคุณให้กับบุคคลที่สาม ยกเว้นในกรณีต่อไปนี้:

  • ได้รับความยินยอมจากคุณ
  • เป็นไปตามกฎหมายหรือคำสั่งของหน่วยงานที่เกี่ยวข้อง
  • มีการใช้ผู้ให้บริการภายนอกที่จำเป็นต้องเข้าถึงข้อมูลบางส่วนในการให้บริการ

4. การจัดเก็บและการรักษาความปลอดภัย

เราจะเก็บข้อมูลของคุณในระยะเวลาที่จำเป็นต่อการให้บริการ และมีมาตรการรักษาความปลอดภัยที่เหมาะสม เช่น การเข้ารหัส และการควบคุมการเข้าถึง

5. สิทธิของผู้ใช้งาน

คุณมีสิทธิในการเข้าถึง แก้ไข หรือขอลบข้อมูลส่วนบุคคลของคุณได้ โดยสามารถติดต่อเราผ่านช่องทางที่ระบุด้านล่าง

6. การเปลี่ยนแปลงนโยบาย

เราขอสงวนสิทธิ์ในการปรับปรุงนโยบายความเป็นส่วนตัวนี้ หากมีการเปลี่ยนแปลง เราจะแจ้งให้คุณทราบผ่านแอปหรือช่องทางอื่นที่เหมาะสม
